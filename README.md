# StockPro

Sistema de controle de estoque desenvolvido em linguagem C para empresas de móveis planejados.

## Funcionalidades

- Cadastro de produtos
- Listagem de estoque
- Entrada de produtos
- Saída de produtos
- Remoção de produtos
- Salvamento em arquivo

## Tecnologias

- Linguagem C
- Manipulação de arquivos
- Structs
- Modularização

## Como executar

gcc main.c src/*.c -o stockpro 
(cria um stockpro.exe)

./stockpro 
(inicia o arquivo stockpro.exe)
